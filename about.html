About Me

조하율

정보보호학을 전공하며 단순한 이론 암기를 넘어, 원리를 이해하고 직접 구현해 보는 것에 매력을 느낍니다.

현재 웹 퍼블리싱부터 리눅스 시스템 관리, 네트워크 프로토콜 분석까지 인프라 전반의 기초 체력을 단단히 다지고 있습니다. 취약점을 찾아내고 방어하는 보안 엔지니어가 되기 위해 매일 배운 것을 기록하고 한 걸음씩 성장하고 있습니다.

기술 스택

Web

HTML, CSS, JavaScript를 활용해 구조가 탄탄한 반응형 웹페이지를 구현합니다.

Linux

vi 에디터 활용, 기본 명령어 다루기, 파일 권한 및 사용자 관리 개념을 이해하고 실습했습니다.

Network

OSI 7계층, TCP/UDP, IP, HTTP, DNS 등 패킷의 흐름과 통신 전반의 개념을 정리했습니다.

Security

시스템 하드닝(Hardening), 접근 통제, 그리고 안전한 서버 보안 설정에 깊은 관심을 가지고 있습니다.

성장 목표

현재

학습한 이론과 트러블슈팅 경험을 블로그와 GitHub에 꼼꼼히 기록 중

단기 목표

다양한 환경에서의 시스템 하드닝 가이드를 분석하고, 나만의 보안 웹 애플리케이션 토이 프로젝트 제작

장기 목표

인프라와 클라우드 환경을 안전하게 설계하고 방어하는 전문 보안 엔지니어로 성장
